# Paso 0
1.- Descargar, instalar `Git`
2.- Descargar, instalar `VSC`
3.- Crear un directorio y abrir VSC
4.- Abrir terminal y probar los comandos:
    git -v == ver version de git
    git config --global user.name "nombre" == agregar nombre de usuario
    git config --global user.email "correo" == agregar correo
5.- Clonar repositorio creado en GitHub:
    git clone (url)

# Paso 1
Cada cambio del proyecto local cuando queremos guardar hacer lo siguiente:
    git add . == prepara los archivos
    git commit -m "comentario del commit"
    git push origin main
